- Breakpoint
Si tratta del punto di interruzzione in cui il contenuto e il layout di un sito web si adatteranno per fornire la migliore esperienza utente possibile su quel determinato dispositivo.
Può funzionare in modi diversi su tre diverse griglie:
Fix grid (colonne fisse e margini flessibili), 
Fluid grid (contenuto flessibile che va da bordo a bordo, le colonne possono cambiare grandezza), 
hybrid grid (Combinazione)

- Bootstrap Breakpoing
Extra small: per schermi di larghezza minore  a 576 pixel, esso è di default.
Small (sm): per schermi di larghezza maggiore o uguale a 576 pixel.
Medium (md): per schermi di larghezza maggiore o uguale a 768 pixel.
Large (lg): per schermi di larghezza maggiore o uguale a 992 pixel.
Extra large (xl): per schermi di larghezza maggiore o uguale a 1200 pixel.
Extra extra large (xxl): per schermi di larghezza maggiore o uguale a 1400 pixel.


- Alert 

                    <div class="alert alert-info" role="alert">
                        Questa è una notifica di esempio!
                    </div>

In Bootstrap, esistono diversi tipi di alert, ognuno con un colore e uno scopo specifico. Ecco un elenco dei principali alert disponibili:

Alert Primary: Utilizzato per messaggi informativi generali. Colore: blu.
Alert Secondary: Utilizzato per messaggi secondari o meno importanti. Colore: grigio.
Alert Success: Indica un'operazione riuscita o un messaggio positivo. Colore: verde.
Alert Info: Utilizzato per messaggi informativi. Colore: azzurro.
Alert Warning: Indica un avviso o una situazione che richiede attenzione. Colore: giallo.
Alert Danger: Indica un errore o un problema. Colore: rosso.
Alert Light: Utilizzato per messaggi leggeri o meno evidenti. Colore: grigio chiaro.
Alert Dark: Utilizzato per messaggi che richiedono un aspetto più forte. Colore: grigio scuro.

- Badge

        <p class="card-text">Descrizione articolo 2 <span class="badge bg-primary">Nuovo</span></p>

Il badge è un componente di Bootstrap utilizzato per evidenziare informazioni importanti, come la novità di un piatto in un menu. Ecco i principali tipi di badge:

Badge Primary: badge bg-primary - Colore blu, usato per informazioni generali.
Badge Secondary: badge bg-secondary - Colore grigio, per informazioni secondarie.
Badge Success: badge bg-success - Colore verde, per indicare successo.
Badge Danger: badge bg-danger - Colore rosso, per avvisi o errori.
Badge Warning: badge bg-warning - Colore giallo, per avvisi.
Badge Info: badge bg-info - Colore azzurro, per informazioni.
Badge Light: badge bg-light - Colore grigio chiaro, per un aspetto leggero.
Badge Dark: badge bg-dark - Colore grigio scuro, per un aspetto forte.
