- Responsive
Unione di: Flexible grids, Fluid images, Media queries

- Breakpoint
Si tratta del punto di interruzzione in cui il contenuto e il layout di un sito web si adatteranno per fornire la migliore esperienza utente possibile su quel determinato dispositivo.
Può funzionare in modi diversi su tre diverse griglie:
Fix grid (colonne fisse e margini flessibili), 
Fluid grid (contenuto flessibile che va da bordo a bordo, le colonne possono cambiare grandezza), 
hybrid grid (Combinazione)

- Bootstrap Breakpoing
Extra small: per schermi di larghezza minore  a 576 pixel, esso è di default.
Small (sm): per schermi di larghezza maggiore o uguale a 576 pixel.
Medium (md): per schermi di larghezza maggiore o uguale a 768 pixel.
Large (lg): per schermi di larghezza maggiore o uguale a 992 pixel.
Extra large (xl): per schermi di larghezza maggiore o uguale a 1200 pixel.
Extra extra large (xxl): per schermi di larghezza maggiore o uguale a 1400 pixel.
