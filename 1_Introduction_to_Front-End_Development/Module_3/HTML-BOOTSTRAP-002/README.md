- Responsive
Combination of: Flexible grids, Fluid images, Media queries

- Breakpoint
This is the breaking point where the content and layout of a website will adapt to provide the best possible user experience on that particular device.
It can work in different ways on three different grids:
Fixed grid (fixed columns and flexible margins), 
Fluid grid (flexible content that goes from edge to edge, columns can change size), 
Hybrid grid (Combination)

- Bootstrap Breakpoints
Extra small: for screens with width less than 576 pixels, it is the default.
Small (sm): for screens with width greater than or equal to 576 pixels.
Medium (md): for screens with width greater than or equal to 768 pixels.
Large (lg): for screens with width greater than or equal to 992 pixels.
Extra large (xl): for screens with width greater than or equal to 1200 pixels.
Extra extra large (xxl): for screens with width greater than or equal to 1400 pixels.


- Alert 

                    <div class="alert alert-info" role="alert">
                        This is a sample notification!
                    </div>

In Bootstrap, there are several types of alerts, each with a specific color and purpose. Here is a list of the main available alerts:

Alert Primary: Used for general informational messages. Color: blue.
Alert Secondary: Used for secondary or less important messages. Color: gray.
Alert Success: Indicates a successful operation or positive message. Color: green.
Alert Info: Used for informational messages. Color: light blue.
Alert Warning: Indicates a warning or situation requiring attention. Color: yellow.
Alert Danger: Indicates an error or problem. Color: red.
Alert Light: Used for light or less evident messages. Color: light gray.
Alert Dark: Used for messages requiring a stronger appearance. Color: dark gray.

- Badge

        <p class="card-text">Article description 2 <span class="badge bg-primary">New</span></p>

The badge is a Bootstrap component used to highlight important information, such as the novelty of a dish in a menu. Here are the main types of badges:

Badge Primary: badge bg-primary - Blue color, used for general information.
Badge Secondary: badge bg-secondary - Gray color, for secondary information.
Badge Success: badge bg-success - Green color, to indicate success.
Badge Danger: badge bg-danger - Red color, for warnings or errors.
Badge Warning: badge bg-warning - Yellow color, for warnings.
Badge Info: badge bg-info - Light blue color, for information.
Badge Light: badge bg-light - Light gray color, for a light appearance.
Badge Dark: badge bg-dark - Dark gray color, for a strong appearance.
