### Module_3 - Bootstrap and React
**Date**: 11/16/2025

Introduction to Bootstrap, CSS frameworks, and basic React concepts.

**Topics**:
- **Bootstrap**
  - Bootstrap basics
  - Responsive grid system
  - Predefined components
  - Infix (responsive breakpoints): `col-sm-6`, `text-md-center`, `d-lg-none`
  - Modifier (style variations): `btn-primary`, `btn-lg`, `alert-danger`, `card-body`
  - [Bootstrap Documentation](https://getbootstrap.com/docs/5.3/getting-started/introduction/)

- **Modern CSS Frameworks**
  - **Bootstrap**: Predefined components, grid system, rapid design
  - **Foundation**: Advanced components, complex projects
  - **Pure.css**: Minimalist, lightweight, performance
  - **Tailwind CSS**: Utility-first, custom design
  - **UIKit**: Lightweight, clean and modular design
  - **MVP.css**: Rapid prototyping without CSS classes

- **React**
  - React is a **library** (not a framework)
  - Virtual DOM functionality
  - Components and rendering

- **Advanced Concepts**
  - **Dependencies**: External packages (React, Tailwind, Bootstrap)
  - **Bundlers**: Tools like Vite, Webpack, Parcel to optimize code
  - **Lazy Loading**: On-demand loading to improve performance

- **JavaScript Libraries with React**
  - **Lodash**: Utility functions (sorting, rounding)
  - **Luxon**: Date and time management
  - **Redux**: Application state management
  - **Axios**: Simplified HTTP requests
  - **Jest**: Automated testing

## üõ†Ô∏è Key Concepts Learned

### HTML5 - Basic Structure
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body>
  <h1>Welcome</h1>
  <a href="page.html">Link</a>
</body>
</html>
```

### CSS - Box Model
```css
.box {
  width: 300px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}
```

### Bootstrap - Grid System
```html
<div class="container">
  <div class="row">
    <div class="col-md-6">Column 1</div>
    <div class="col-md-6">Column 2</div>
  </div>
</div>
```

### Bootstrap - Components
```html
<!-- Navbar -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Brand</a>
</nav>

<!-- Card -->
<div class="card">
  <div class="card-body">
    <h5 class="card-title">Title</h5>
    <p class="card-text">Card text</p>
    <a href="#" class="btn btn-primary">Action</a>
  </div>
</div>
```

## üöÄ Bundlers and Optimization

### Bundlers
Tools that bundle and optimize project files:
- **Vite**: Fast, modern
- **Webpack**: Powerful, configurable
- **Parcel**: Zero-config

### Lazy Loading
Technique to load code only when needed:
- Reduces initial loading time
- Improves performance
- Loads less code at startup
